Detailed Description

형태에 관한 모든 정보를 담는 구조체의 템플릿

Note
변경가능한 상태로 인덱스와 관련된 값이나 std::vector 등의 길이를 변경할 수 있음. kiwi::KiwiBuilder에서 사용한다. baked = true는 변경 불가능한 상태로 인덱스는 모두 포인터로, std::vector는 FixedVector로 변경되어 수정이 불가능한 대신 각 값에 효율적으로 빠르게 접근 가능하다. 이 상태는 kiwi::Form이라는 타입의 부모클래스로 쓰이며, kiwi::Kiwi 내 실제 형태소 분석 단계에 쓰인다.
